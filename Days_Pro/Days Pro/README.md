# 前言
之前想填的倒数日的坑终于填了。
仿自倒数日应用”Days Matter”。

# 说明
* 事件名称请尽量简洁明了。由于系统限制，日期选择区间将限制为1700-2099年，天数为99999天。
* 主题色可以分别选择过去(左侧)和将来(右侧)的颜色。
* 节假日模式是指自动显示为下一个法定节假日，不再显示自定义事件。
